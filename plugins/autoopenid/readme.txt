Плагин AutoOpenId (версия 2.2.3) для LiveStreet 1.0

Описание
Позволяет пользователям авторизоваться/зарегистрироваться через ВКонтакте, Facebook, Twitter, Gmail, Yandex, Mail.ru, Одноклассники.
При возможности автоматически загружает данные пользователя - имя, емайл, пол, аватар и фото.
Есть возможность привязки аккаунтов к уже существующим пользователям.
Данная версия НЕ поддерживает авторизацию по морально устаревшему протоколу OpenID (Рамблер, ЖЖ и другие), поддерживается только OAuth версий 1 и 2.

ВНИМАНИЕ! Плагин требует версию PHP не ниже 5.3


УСТАНОВКА
1. Скопировать плагин в каталог /plugins/
2. Через панель управления плагинами (/admin/plugins/) запустить его активацию. В панели управления он будет называться AutoOpenId.
3. Выполнить настройки плагина в файле /plugins/autoopenid/config/config.php

Для каждого сервиса авторизации необходимо создать свое приложение в соц сетях и прописать необходимые данные в конфиге (ID и секретный ключ)


ОБНОВЛЕНИЕ С ВЕТКИ 1.*
0. Деактивировать и удалить плагин.
1. Загрузить новый плагин.
2. Активировать плагин.
4. Для конвертации существующих связей пройти по адресу /login/autoopenid/migration/  (возможен перенос только аккаунтов Facebook, ВКонтанкте и Одноклассники)
5. ВНИМАНИЕ! Перенос связей протокола OpenID на новую версию не возможен!
